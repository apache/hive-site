# Apache Hive : Books, Blogs & Talks

* **Books:**
* + *[Programming Hive](http://shop.oreilly.com/product/0636920023555.do)* by Edward Capriolo, Dean Wampler, and Jason Rutherglen – O'Reilly Media, 2012
